# Course profiles

Select a course profile after applying the common path and evidence rules.

| Profile | Primary learning loop | Useful modules |
|---|---|---|
| Quantitative | Signal recognition -> method choice -> worked steps -> variation | Formula reference, dependency graph, calculation checks |
| Programming | Predict -> complete -> debug -> implement independently | Runnable examples, tests, error diagnosis |
| Law | Issue -> rule -> facts -> application -> conclusion | Element checklist, contrasting cases |
| Humanities | Claim -> evidence -> comparison -> counterargument -> timed response | Timeline, outline recall, rubric-based self-check |
| Language | Context input -> retrieval -> production -> correction -> reproduction | Spaced vocabulary, speaking/writing prompts |
| Medicine | Presentation -> differential -> evidence -> decision -> safety check | Case progression, red flags, uncertainty |
| Laboratory | Sequence -> risk -> observation -> exception handling | Procedure cards, safety checks, troubleshooting |

Do not add quantitative-course modules to every course. Choose only the modules that support the course's assessment behavior.
