多元函数是一元函数的推广：z=f(x,y)表示两个自变量x,y确定一个因变量z。定义域通常是平面上的一个区域（闭区域/开区域）。

💡 几何直观：z=f(x,y)在三维空间中是一张曲面。例如z=x²+y²是旋转抛物面（碗状），z=√(1-x²-y²)是上半球面。
极限定义：$\lim_{(x,y)\to(x_0,y_0)} f(x,y) = A$ 当且仅当沿任意路径趋近函数值都趋于A
⚠️ 常见错误：沿两条不同路径得到不同极限，则极限不存在（最常用的反证法！）。比如沿y=kx方向极限与k有关，则极限不存在。
⚡ 连续性：多元初等函数在其定义域内都是连续的——所以考试时求极限，直接代入即可（只要代入后是有定义的点）。

偏导数就是"偏向某一个变量"的导数：求f对x的偏导f_x，就把y当作常数，按一元函数求导法则对x求导。

💡 几何直观：f_x(x₀,y₀)是曲面z=f(x,y)在点(x₀,y₀)处沿x轴方向切线的斜率；f_y同理沿y轴方向。
偏导定义：$f_x(x_0,y_0) = \lim_{\Delta x \to 0} \frac{f(x_0+\Delta x,y_0)-f(x_0,y_0)}{\Delta x}$
📝 典型例题1：求一阶偏导 📌 母题
题目：设 $f(x,y) = x^2y + \sin(xy) + e^x\ln y$，求 $f_x, f_y$。
💡 怎么想到的？求f_x时把y当常数，逐项对x求导；求f_y时把x当常数，逐项对y求导。sin(xy)对x求导要用链式法则：外层cos(xy)乘以内层对x的导数y。
① 求f_x（y视为常数）：(x²y)'_x=2xy；[sin(xy)]'_x=cos(xy)·y；[e^x ln y]'_x=e^x ln y
结果：f_x = 2xy + y·cos(xy) + e^x·ln y
② 求f_y（x视为常数）：(x²y)'_y=x²；[sin(xy)]'_y=cos(xy)·x；[e^x ln y]'_y=e^x·(1/y)
结果：f_y = x² + x·cos(xy) + e^x/y
📚 从这道题学到：①"固定其他变量"是核心口诀；②复合函数用链式法则，别忘了乘以内层导数；③ln y对y求导是1/y（不要写成y）；④高阶偏导：f_xy先对x再对y，混合偏导连续时f_xy=f_yx（克莱罗定理）。

变式：求 $f(x,y)=x^3y^2+e^{xy}$ 在点(1,0)处的 $f_x(1,0)$ 和 $f_y(1,0)$。

答案：f_x(1,0)=0（先代y=0，f(x,0)=1，f_x=0）；f_y(1,0)=1（先代x=1，f(1,y)=y²+e^y，f_y=2y+e^y，代入y=0得1）。技巧：先代值再求导更简单！

📝 典型例题2：高阶偏导 📌 常考
题目：设 $z = x^3y^2 - 3xy^3 + xy + 1$，求 $\frac{\partial^2 z}{\partial x^2}, \frac{\partial^2 z}{\partial x\partial y}, \frac{\partial^2 z}{\partial y^2}$。
💡 怎么想到的？先求一阶偏导，再对结果求偏导。混合偏导∂²z/∂x∂y先对x求导再对y求导。
① 求一阶偏导：∂z/∂x = 3x²y² - 3y³ + y；∂z/∂y = 2x³y - 9xy² + x
已得到一阶偏导
② 二阶偏导：∂²z/∂x² = 6xy²；∂²z/∂x∂y = 6x²y - 9y² + 1；∂²z/∂y² = 2x³ - 18xy
结果如上，验证∂²z/∂x∂y = ∂²z/∂y∂x = 6x²y - 9y² + 1 ✓
📚 要点：混合偏导在连续的条件下与求导顺序无关，这可以用来验证你的计算是否正确。
⚠️ 易错点：①对x求偏导时忘记y是常数，把y也求导了；②链式法则漏乘内层导数（如sin(xy)对x求导漏乘y）；③高阶混合偏导顺序搞错符号。

全微分描述函数在两个方向同时变化时的总变化量近似。

全微分公式：$dz = \frac{\partial z}{\partial x}dx + \frac{\partial z}{\partial y}dy = f_x dx + f_y dy$

可微的充分条件：f_x和f_y都存在且连续 → f可微。

链式法则（复合函数求导）

若 z=f(u,v), u=u(x,y), v=v(x,y)：$\frac{\partial z}{\partial x} = \frac{\partial z}{\partial u}\frac{\partial u}{\partial x} + \frac{\partial z}{\partial v}\frac{\partial v}{\partial x}$
口诀：沿线相乘，分线相加（画树形图！）
✅ 隐函数求导：F(x,y,z)=0确定z=z(x,y)，则 ∂z/∂x = -F_x/F_z，∂z/∂y = -F_y/F_z。记住负号！

极值求解分两步：①找驻点（f_x=0且f_y=0）；②用二阶偏导判别法判定。

判别法：设 $A=f_{xx}, B=f_{xy}, C=f_{yy}$
(1) $AC-B^2 > 0$：是极值，A<0极大，A>0极小
(2) $AC-B^2 < 0$：不是极值（鞍点）
(3) $AC-B^2 = 0$：无法判断，需另法讨论
📝 典型例题：求f(x,y)=x³-4x²+2xy-y²的极值 📌 母题
题目：求函数极值。
💡 思路：先求驻点，再用AC-B²判别。
① 求驻点：f_x=3x²-8x+2y=0，f_y=2x-2y=0 → y=x，代入得3x²-6x=0 → x=0或x=2，驻点(0,0)和(2,2)
② 判别(0,0)：A=f_xx=6x-8=-8, B=2, C=-2；AC-B²=16-4=12>0, A<0 → 极大值f(0,0)=0
③ 判别(2,2)：A=12-8=4, B=2, C=-2；AC-B²=-8-4=-12<0 → 不是极值
📚 要点：条件极值用拉格朗日乘数法：构造L=f+λφ，令L_x=L_y=L_λ=0。

二重积分的几何意义是曲顶柱体体积。计算核心思想：化为累次积分。

直角坐标：$\iint_D f(x,y)\,dA = \int_a^b\!\!\int_{y_1(x)}^{y_2(x)} f(x,y)\,dy\,dx$（X型）
或 $= \int_c^d\!\!\int_{x_1(y)}^{x_2(y)} f(x,y)\,dx\,dy$（Y型）
📝 典型例题1：X型区域 📌 母题
题目：计算 $\iint_D xy\,dA$，其中D由 y=x, y=0, x=1 围成。
💡 怎么想到的？先画积分区域D：是x从0到1之间，y从0到x的三角形。选择X型（先y后x）更方便：x∈[0,1], y∈[0,x]。
① 确定积分限：X型：x从0到1，对每个x，y从0到x（下边界y=0，上边界y=x）
∫₀¹∫₀ˣ xy dy dx
② 内层（对y积分，x当常数）：∫₀ˣ xy dy = x·[y²/2]₀ˣ = x·(x²/2) = x³/2
结果：x³/2
③ 外层（对x积分）：∫₀¹ x³/2 dx = (1/2)·[x⁴/4]₀¹ = 1/8
最终结果：1/8
📚 从这道题学到：①先画区域图再定限——这步绝不能省；②内层积分把外变量当常数；③积分结果是一个数，不再含有x,y；④交换积分次序有时能大幅简化计算（当f对x难积但对y好积时）。

变式：交换 ∫₀¹∫_y¹ e^{x²} dx dy 的积分次序并计算。

提示：原次序对x积分e^{x²}积不出来！换Y型→X型：区域D是0≤y≤x≤1的三角形，改为∫₀¹∫₀ˣ e^{x²} dy dx = ∫₀¹ xe^{x²} dx = (e-1)/2。这是"必须交换次序"的经典题！

📝 典型例题2：利用对称性简化 ⚡ 秒杀技
题目：D为x²+y²≤1（单位圆），求 $\iint_D (3x+2y+5)\,dA$。
💡 怎么想到的？D关于x轴和y轴对称！奇函数在对称区域上积分为0。
① 分项利用对称性：3x是x的奇函数（D关于y轴对称）→积分=0；2y是y的奇函数（D关于x轴对称）→积分=0
只剩∫∫_D 5 dA = 5×面积 = 5π
📚 秒杀技巧：看到对称区域，先看被积函数是否有奇偶性——可以直接干掉某些项，省去大量计算！
⚠️ 常见错误：①积分限写成常数导致错误（必须画区域图！）；②累次积分的内外层搞反；③忘记先积变量在当常数时"不能出现在积分结果里"。

当积分区域是圆、扇形、环形，或被积函数含x²+y²时，极坐标是利器。

⚡ 核心公式：x=rcosθ, y=rsinθ，雅可比行列式J=r，故 $dA = r\,dr\,d\theta$。
⚠️ 这个r是挂科率最高的考点，千万别漏！
📝 典型例题：∫∫_D e^{-x²-y²} dσ（D为单位圆） 📌 极坐标经典题
题目：直角坐标下e^{-x²}积不出来，必须用极坐标！
① 变换：x²+y²=r²，dσ=rdrdθ，θ∈[0,2π], r∈[0,1]
② 计算：=∫₀^{2π}dθ∫₀¹ e^{-r²}·r dr = 2π·[-1/2·e^{-r²}]₀¹ = π(1-e^{-1})
📚 要点：别忘了r！若漏了r，内层积分∫₀¹ e^{-r²}dr根本积不出来——这本身就是一个信号。

三重积分是二重积分的推广，三种坐标系选择：

① 直角坐标：$dV = dxdydz$
② 柱面坐标：$x=r\cos\theta, y=r\sin\theta, z=z$，$dV = r\,dr\,d\theta\,dz$（多一个r）
③ 球面坐标：$x=r\sin\varphi\cos\theta, y=r\sin\varphi\sin\theta, z=r\cos\varphi$，$dV = r^2\sin\varphi\,dr\,d\varphi\,d\theta$（多r²sinφ）
✅ 选择坐标系口诀：区域是圆柱/旋转体→柱面坐标；是球/锥→球面坐标；其余→直角坐标。投影法（"先一后二"）最常用。

第一类曲线积分（对弧长）：∫_L f(x,y) ds，物理意义是曲线的质量。

第二类曲线积分（对坐标）：∫_L Pdx+Qdy，物理意义是变力沿曲线做功。

格林公式：设L是正向（逆时针）闭曲线，D为L围的区域，P,Q有连续一阶偏导：
$\oint_L P\,dx + Q\,dy = \iint_D \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right) d\sigma$
⚡ 格林公式三个条件：①闭合；②正向（逆时针）；③P,Q在D内有连续一阶偏导。不闭合时"补线"再用格林公式；有奇点时"挖洞"。
✅ 路径无关条件：在单连通区域内 ∂Q/∂x=∂P/∂y ⇔ 积分与路径无关 ⇔ 存在原函数u使du=Pdx+Qdy。
